# GESTURE-BASED-MUSIC-CONTROL-SYSTEM-
A gesture-controlled music system that lets users play, pause, skip, or adjust volume using simple hand gestures without Arduino and sensors.
today’s world, touchless control is becoming essential in smart technology.
•This project introduces a Gesture-Based Music Control System, allowing users to
control music playback using hand gestures detected via webcam.
•It provides a contact-free, user-friendly, and interactive way to enjoy music.
